# models/

3D assets (`.glb` / `.gltf`) loaded into the pyramid.

Guidelines:
- Low-poly, bright emissive materials — pyramid eats subtle detail
- Pure-black background renders best
- Keep individual files < 5 MB so they load instantly during voice commands

Hero models for the demo (to be locked Tue 6/23):
- [ ] Microsoft hardware product (Surface / Xbox / HoloLens)
- [ ] Microsoft building (B7 / campus)
- [ ] "Wow" model (molecule / engine / heart / etc.)

Sources:
- Sketchfab (free + properly licensed)
- Internal MS 3D library
- Blender (build simple ones in ~30 min)
- Polycam / iPhone LiDAR scans
