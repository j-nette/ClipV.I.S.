# ClipV.I.S.

> *Clippy but they joined the Avengers.*

**Bring the thing you're talking about into the room.**

A physical Pepper's Ghost hologram pyramid for your meeting table. Say *"Clippy, show me the Surface Pro 11"* — a 3D model materializes inside the pyramid, rotating, visible from all sides, with live specs pulled from Microsoft Fabric via an Azure AI Foundry agent. Slideshows show pictures of things. ClipV.I.S. shows the **thing**.

Microsoft Intern Hackathon 2026.
